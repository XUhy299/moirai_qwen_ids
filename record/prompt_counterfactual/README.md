# WADI MOIRAI-Qwen Prompt 反事实消融

## 实验问题与边界

本实验检验固定工控过程 Prompt 是否稳定优于通用、最简和错误领域 Prompt。所有运行均使用 WADI 验证集单攻击事件的 support→query 开发协议，不是正式测试结果，也不能证明跨攻击泛化。

- 窗口与层：`L=patch_size=64`，显式 MOIRAI Base 第12层。
- 表示：按正常训练集剔除恒定变量，保留 80 个变量 Token。
- 模型：DirectProjector + 冻结 Qwen3-0.6B + 二分类头。
- 损失：`classifier_loss_weight=1`，`vocab_loss_weight=0`。
- 随机种子：2026、2027、2028；每个 Prompt 均按开发 PR-AUC 选择 epoch。
- 开发集：6364 个 stride=8 端点，其中异常 85 个；标签为 `Y[window_end]`。
- 这些运行发生在后续 DTT 代码引入测试 X 之前；产物均为 80 变量协议，且没有读取 WADI 正式测试数组。

正确过程 Prompt 与错误过程 Prompt 的 Qwen 前缀/后缀 token 数均为 `40+19`，因此这组关键对照固定了文本长度。通用和最简 Prompt 更短，主要作为删减对照，仍存在长度与内容同时变化的限制。

## 发现

1. 正确工控 Prompt 没有稳定优于错误过程 Prompt；错误 Prompt 的平均 PR-AUC 反而高 0.0180，并在 seed 2027/2028 领先。当前证据不支持“Qwen 正在利用正确工控知识推理”。
2. 最简 Prompt 的平均 PR-AUC 最高，但标准差 0.1198，三个 seed 从 0.3545 降到 0.1151，不能仅凭均值将其锁定为最终 Prompt。
3. 错误 Prompt 的 PR-AUC 波动最小且 FPR 最低，说明文本可能主要改变冻结 Qwen 的表示几何、位置和优化轨迹，而非按语义正确性发挥作用。
4. 用户原始 `实验记录/结果记录.md` 中错误 Prompt seed 2026 的 FPR 写为 0.0067；依据所选 PR-AUC epoch 的 `history.json`，正确值是 0.0201。正式记录采用机器产物值。

## 结论与后续使用

Prompt 消融已经完成，但它否定了“正确工控知识 Prompt 带来稳定收益”的论点。论文中可将这一结果作为诚实的反事实分析：冻结 LLM 对连续时序 Token 有非线性迁移价值，不等于它理解了给定工控知识。

进入锁定验证前，不应继续搜索更多措辞。若需要一个语义可接受的主配置，优先在最简 Prompt 与通用 Prompt 之间基于独立协议选择；错误 Prompt 只能作为反事实，不能作为最终方法。
